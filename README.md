# react_native_paper_example
Jumpstart your react-native app development using this boilerplate of react-native-paper with simple Material Design.

## How to install:
    1. Clone: git clone https://github.com/ferdinandduterte/react_native_paper_example.git
    2. Yarn installation of packages: yarn install
    3. Linking react-native-vector-icons: yarn react-native link react-native-vector-icons
    4. Running app on android: yarn react-native run-android
